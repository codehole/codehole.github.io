从App跳转App

从Safari跳到APP是通过打开特定的URL，那么从 APP（A）跳转APP（B）是不是也要打开特定的URL呢？

答案是肯定的，那么怎么打开这个特定的URL呢，其实在UIApplication里面有这么一个方法openURL，如下

[[UIApplication shareApplication] openURL:url]

例如

打开qq,
[[UIApplication sharedApplication] openURL:[NSURL URLWithString:@”mqq://“]];

如果是模拟器，没有qq, 可以打开短信
[[UIApplication sharedApplication] openURL:[NSURL URLWithString:@”sms://“]];

关于URL Schemes

1, URL Schemes 有两个单词：

URL，我们都很清楚，http://www.apple.com 就是个 URL，我们也叫它链接或网址；
Schemes，表示的是一个 URL 中的一个位置——最初始的位置，即 ://之前的那段字符。比如 http://www.apple.com 这个网址的 Schemes 是 http。
2, iOS 上的 URL Schemes 因为没有统一标准，所以非常难猜，通过猜来获取 iOS 应用的 URL Schemes 是不现实的
